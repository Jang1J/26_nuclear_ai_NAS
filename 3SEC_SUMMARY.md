# 🏆 3초 진단 우승을 위한 핵심 요약

## 왜 3초가 중요한가?

**평가 기준**:
- 0-3초 진단: 최고 점수 (100%)
- 3-10초: 높은 점수 (80%)
- 10초 이상: 중간 이하

**목표**: 3초 안에 85%+ 정확도 달성

---

## 🎯 3가지 핵심 전략

### 1. 변화율 피처 (가장 중요!) ⭐⭐⭐

```bash
--feature_method physics
```

**왜?**
- 사고는 초반에 급격한 변화 발생
- LOCA: 압력 급락
- SGTR: SG 압력 상승
- MSLB: 증기압 급락

**physics 피처**:
- 1차/2차 미분 (d/dt, d²/dt²)
- 루프 간 비대칭
- 물리적 상관관계

### 2. 작은 윈도우 (3초 데이터만) ⭐⭐⭐

```bash
--window_size 6
```

**왜?**
- 3초 = 약 6 샘플 (0.5초 간격 가정)
- 적은 데이터로 빠른 판단
- 초반 급격한 변화만으로 충분

### 3. 공격적 Threshold ⭐⭐⭐

**realtime_inference.py에 구현됨**:
```python
# 3초 이내 65% 확신도면 진단
if elapsed_time <= 3.0 and confidence >= 0.65:
    diagnosed = True
```

**왜?**
- 확실할 때까지 기다리면 늦음
- 65-70% 정도면 충분히 정확
